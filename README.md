# SuperCube (WebGL Deployment)

Dieses Repository enthält den fertigen **Unity WebGL Build** von **SuperCube** und wird automatisch über GitHub Pages gehostet.

🚀 **Jetzt im Browser spielen:** [https://michdo93.github.io/SuperCube/](https://michdo93.github.io/SuperCube/)

---

## Verwandte Repositories

* ⚙️ **Backend (FastAPI):** [Michdo93/SuperCube-Backend](https://github.com/Michdo93/SuperCube-Backend) – Verwaltet die globalen Highscores auf Render.com.
* 🛠️ **Unity Quellcode:** [Michdo93/SuperCube-Unity-Project](https://github.com/Michdo93/SuperCube-Unity-Project) – Die editierbaren Projektdateien für den Unity Editor.

## Features & Integration

* **HTML5 / WebGL:** Läuft direkt im Browser ohne Plugins.
* **Live-Highscore-System:** Sendet und empfängt Highscores in Echtzeit über eine REST-API an das FastAPI-Backend.

## Deployment auf GitHub Pages

Wenn du dieses Spiel selbst hosten möchtest:
1. Forke dieses Repository.
2. Gehe in die Einstellungen (**Settings**) -> **Pages**.
3. Wähle den `main`-Branch (oder den entsprechenden Ordner) als Quelle aus und speichere.
